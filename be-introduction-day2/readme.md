#Project description
```
Project ini dibuat pada tanggal 5 Maret 2021 saat kelas induksi hari ke-2
Class Person dan Class HelloController adalah hasil dari kelas induksi day 2

class lainnya adalah hasil dari pengerjaan saya dalam rangka memenuhi tugas dari kelas induksi day2.
saya membuat aplikasi sederhana untuk menyimpan dan mengambil data siswa.
aplikasi ini dapat 
1. menyimpan data siswa (http://localhost:8080/api/student/v1/create) post
2. melihat semua data siswa (http://localhost:8080/api/student/v1/get-students) get
3. mencari siswa berdasarkan nama (http://localhost:8080/api/student/v1/get-student/{name}) get
4. mencari siswa berdasarkan kelas (http://localhost:8080/api/student/v1/search-students-by-grade?grade=11) get
5. mencari siswa berdasarkan jurusan(http://localhost:8080/api/student/v1/search-students-by-major?major=science) get
6. mencari siswa berdasarkan jurusan dan kelas(http://localhost:8080/api/student/v1/search-students-by-major-and-grade) post
```
